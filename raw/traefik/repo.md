Title: GitHub - traefik/traefik: The Cloud Native Application Proxy

Traefik is the cloud native application proxy. The repository contains the main Traefik codebase along with docs, command-line tooling, internal packages, integration tests, and the web UI.

Repository highlights:
- Cloud native application proxy
- docs, cmd, internal, pkg, integration, and webui directories
- large, active open source project with extensive history
- foundation for routing, ingress, and traffic management use cases
