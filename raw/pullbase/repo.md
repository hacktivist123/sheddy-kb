Pullbase is an open source GitOps project for Linux servers. It manages packages, services, and configuration files declaratively from Git, with agents that pull desired state and reconcile drift on managed servers.

Repository highlights:
- Git as source of truth
- pull-based model for servers, VMs, and bare metal
- drift detection and auto-reconciliation
- dashboard, API, and Git monitor in the server
- agents that report status back to the central service
