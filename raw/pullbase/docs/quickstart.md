Title: Quickstart

The quickstart walks through launching Pullbase with Docker, bootstrapping an admin account, opening the dashboard, and adding agents or real servers. It reinforces the evaluation-to-production path and points to production deployment and configuration reference docs.

Key steps:
- docker compose up
- verify the health endpoint
- bootstrap admin credentials
- open the dashboard
- create a server and install the agent
