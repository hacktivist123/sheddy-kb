Title: What is Pullbase?

Pullbase is presented as GitOps for Linux servers: manage packages, services, and configuration files using Git as the source of truth. The docs emphasize pull-based operation, drift detection, rollbacks, and support for Linux fleets across VMs, bare metal, and cloud servers.

Key ideas:
- define desired state in config.yaml
- agents pull and apply changes
- monitor sync and drift from a dashboard
- use Git history for rollback and auditability
