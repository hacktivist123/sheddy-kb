# Sheddy KB

An autonomous, self-maintaining knowledge base compiled from raw captures, summary notes, and concept articles.

## Layers

- [Source index](source-index.md)
- [Summary layer](summaries/index.md)
- [Concept index](concepts/index.md)

## Operating loop

1. Capture material into raw/
2. Preserve provenance and source links
3. Summarize each source cluster
4. Write concept pages that merge related sources
5. Add backlinks from concepts back to summaries and sources
6. Revisit and expand the graph as new raw material arrives
