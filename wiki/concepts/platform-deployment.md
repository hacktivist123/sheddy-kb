# Platform.sh and cloud deployment

This cluster connects the deployment-focused writing across Platform.sh, Cloud Foundry, and source-ops patterns. It is the bridge between app code and the operational environment that hosts it.

## Core ideas

- Platform.sh as a deployment platform for Gatsby, Next.js, Python, and Go
- Multi-app and headless CMS layouts
- Source operations as a deployment-time capability
- Cloud Foundry deployment for a full-stack Node.js app
- Cloud-native context around how applications are packaged and shipped
