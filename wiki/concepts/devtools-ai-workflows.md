# Dev tools and AI workflows

This cluster collects AI-assisted writing, coding, and automation workflows, along with the git and debugging techniques that support them.

## Core ideas

- Working with AI tools in everyday development
- Building small automations and bots with webhooks and GitHub APIs
- Git hygiene and fixup commits
- Using GitHub Desktop in environments that are not GitHub-first
- Debugging and tooling as part of the workflow, not an afterthought
