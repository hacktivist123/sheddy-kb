# React and frontend workflows

The frontend cluster focuses on React foundations, component patterns, styling, API consumption, and browser tooling. It also captures adjacent build-tool and debugging notes that support a modern frontend workflow.

- Hooks, higher-order components, and reusable UI composition
- Styling strategies for React applications
- REST API consumption from the browser
- Bundle-size reduction and debugger workflows for frontend development
