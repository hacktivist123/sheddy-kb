# sheddy-kb

An LLM-managed personal knowledge base.

This repository is intended to store source material, raw captures, and synthesized notes that an LLM can help maintain over time. The workflow is simple:

- capture sources into raw/
- preserve provenance and source URLs
- synthesize durable notes from the source material
- keep the knowledge base easy to search, revise, and expand

Current indexing priority:

1. Pullbase sources
2. Traefik sources
3. Additional personal and work references as needed

Reference inspiration:

- Andrej Karpathy on LLMs and context management: https://x.com/karpathy/status/2039805659525644595
- Anthropic on autonomous knowledge base setup: https://x.com/anthropicai/status/2039749628737019925
